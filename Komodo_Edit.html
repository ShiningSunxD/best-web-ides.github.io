Komodo IDE

Komodo Edit — это кросс-платформенный текстовый редактор, который построен на движке Mozilla, поддерживает большое количество языков. Проект является полностью открытым и распространяется под лицензией MPL (Mozilla Public License). Разработка проекта ведется на GitHub. Как и у всех, есть авто-дополнение со справочной информацией, поддержка плагинов, макросы, сниппеты. Главная фича, которая мне понравилась, это удаленное редактирование кода, подобно Sublime если прикрутить к нему плагины для работы по SFTP. В восьмой версии также появился визуальный скроллинг, аналогично тому, который есть в Sublime.

Основные функции

  1. Выделение синтаксиса
  2. Автокоррекция стандартных ошибок (незакрытые скобки, кавычки, теги).
  3. Инструменты отладки.
  4. Модульное тестирование.
Komodo Edit — многофункциональный редактор. Его можно использовать для крупных проектов. Инструмент является полнофункциональной средой разработки. Его возможности намного шире, чем у большинства аналогов.

Преимущества

  1. Проработанный и удобный интерфейс. Пользователь может настроить параметры отображения с учетом персональных предпочтений.
  2. Во встроенной библиотеке представлено множество элементов для автоматической вставки части кода.
  3. Информация отображается в виде древовидной структуры. Это упрощает работу с объектами большого объема.
  4. Удобный механизм поиска позволяет находить информацию не только в открытом документе, но и в неактивных вкладках.
  5. Предлагаются различные дополнения для расширения возможностей сервиса.
